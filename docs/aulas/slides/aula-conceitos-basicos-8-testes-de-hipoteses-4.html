Estatística não paramétrica

  • Testes paramétricos
    • Possuem várias exigências
      • Distribuição normal
      • Homogeneidade de variâncias
    • Hipóteses feitas sobre parâmetros (Ex. média)
  • Testes não-paramétricos
    • Não possuem exigências de distribuição
    • Não substituem os testes paramétricos
      • As hipóteses são diferentes
      • Podemos aplicar caso as premissas para paramétricos sejam seriamente transgredidas

Estatística não paramétrica

  • Testes não-paramétricos
    • Não são calculadas com base nos dados originais, mas nos postos
  • Postos
    • Ordene os dados
    • Atribua a cada valor um número de ordem
    • Empates
      • Atribua as médias dos postos que receberiam.
    • A soma dos postos é sempre igual a
    \[ \frac{n(n-1)}{2} = \frac{8(8+1)}{2} = 36 \]

Testes de significância para duas amostras

  • Teste de Mann- Whitney
    • Finalidade
      • Verificar se duas populações tem a mesma distribuição
      • Alternativa ao teste t para amostras independentes
    • Condições
      • Amostras são independentes
      • Amostras pequenas e/ou pressuposições para t muito comprometidas
      • Observações medidas em escala ordinal ou numérica

Testes de significância para duas amostras

  • Teste de Mann- Whitney
    • Estabelecemos o nível de significância
    • Separe os grupos 1(n1) e 2(n2)
    • Junte os dois em um grupo e determine os postos
    • Calcule \(\sum{R_1}\) e \(\sum{R_2}\)
    • Calcule a estatística de Mann-Whitney
      • Se \(n_1 = n_2\) – utilize o grupo de menor \(\sum{R}\)
      • Se \(n_1 \neq n2\) – calcule para os dois e escolha o menor U
      \[ \sum{R_1} - [\frac{n_1(n_1+1)}{2}] \]

Testes de significância para mais de duas amostras

  • Kruskal-Wallis
    • Finalidade
      • Testar se varias populações tem a mesma distribuição
      • Alternativa para Anova univariada (Anova by ranks test)
    • Condições
      • Amostras pequenas e/ou pressuposições muito comprometidas
      • Observações medidas em escala ordinal ou numérica
      • Se houverem somente 3 grupos, necessário haver pelo menos 5 casos

Testes de significância para mais de duas amostras

  • Kruskal-Wallis

    • A lógica é semelhante ao teste de Mann-Whitney

      • São executados para k grupos
      • Estabelece postos
