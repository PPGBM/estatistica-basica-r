Graficos em R

  • Modificando os parâmetros gráficos
    • A função par()
      • Permite configurar a área gráfica
      • Número de gráficos, margens, tamanho, etc…
par(mfrow = c(1,2))
x <- 1:10
plot(x,y)
barplot(x)
# Abra a ajuda para verificar outros parâmetros

?par
