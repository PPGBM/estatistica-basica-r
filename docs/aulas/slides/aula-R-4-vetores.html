Sequencias, distribuições e amostragens

  • “:” - nos ja vimos este
  • “seq()”
    • cria sequências de números seq(inicio, fim, intervalo)
  • “rep()”
    • repete um valor ou um vetor > rep(valor, vezes)
  • “runif()”
    • gera números aleatórios de uma distribuição uniforme > runif(n, min, max)
seq(1, 10)
seq(1, 10, 2)

numeros <- c(1:3)
rep(numeros, 3)
rep(numeros, 1:3)

fator <- c("A", "B", "C")
rep(fator, each = 5)

runif(100, 1, 25)
