Boas práticas para páginas do site

  • Usar format: html
  • Evitar slides (revealjs) em páginas
  • Texto mais explicativo e menos fragmentado
  • Tabelas e figuras sempre com legenda
  • Blocos semânticos para guiar a leitura

Diferença essencial: site × slides

  • Site (HTML) → leitura contínua, explicação detalhada
  • Slides (reveal.js) → síntese, apoio visual, pouco texto

Nunca misturar os dois formatos no mesmo arquivo.
