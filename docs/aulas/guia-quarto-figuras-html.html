Guia detalhado – Figuras em páginas Quarto HTML

Conceito geral

Em páginas HTML geradas com Quarto, figuras são usadas para: - ilustrar conceitos - apresentar gráficos - documentar resultados - apoiar explicações longas

Diferente de slides, o foco é leitura confortável e organização vertical.

Inserção básica de imagens

Imagem simples

![](figuras/exemplo.png)
  • Usa o tamanho original da imagem
  • Não recomendado sem controle de largura

Imagem com legenda (recomendado)

![Legenda da figura](figuras/exemplo.png)
  • A legenda aparece abaixo da imagem
  • Essencial para páginas didáticas

Controle de tamanho

Largura em porcentagem (recomendado)

![](figuras/exemplo.png){width=60%}

Valores comuns: - 40% → imagem pequena - 60% → padrão - 80% → destaque

Largura em pixels

![](figuras/exemplo.png){width=600px}

Útil quando: - precisa de tamanho fixo - mantém proporção conhecida

Altura (uso cuidadoso)

![](figuras/exemplo.png){height=400px}

⚠️ Pode distorcer a imagem se usada sozinha.

Largura + altura

![](figuras/exemplo.png){width=600px height=400px}

⚠️ Use apenas quando conhecer a proporção exata.

Boas práticas para HTML

  • Texto explicativo antes e depois da figura
  • Figuras sempre com legenda
  • Evitar imagens muito grandes
  • Usar colunas apenas quando necessário
  • Testar em telas pequenas

Erros comuns

  • Imagens sem legenda
  • Imagens enormes quebrando o layout
  • Muitas figuras seguidas sem texto
  • Uso excessivo de colunas

Regra de ouro para páginas

A figura deve complementar o texto,
nunca substituí-lo.
