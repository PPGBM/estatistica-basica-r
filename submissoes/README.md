# Submissões de Exercícios

Esta pasta armazena os arquivos enviados pelos alunos
por meio do formulário do site da disciplina.

## Organização

- Cada lista possui sua própria subpasta
- Os arquivos são nomeados automaticamente ou conforme instrução
- Não editar arquivos manualmente nesta pasta

## Atenção

Esta pasta é usada por scripts de correção automática.
